# Gestion-des-Devis-et-Factures-Spring-Boot-MySQL-REST-API-
Créer une application web pour gérer les devis, factures, clients et produits d’une entreprise. Technologies : Backend : Spring Boot, Spring Data JPA, Spring MVC, MySQL Frontend : Bootstrap/CSS + JavaScript (Fetch API) Build : Maven Optionnel : Export PDF avec iText / JasperReports
